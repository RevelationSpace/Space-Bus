Detailed Description

An SBLP hardware-side implementation for the ATTiny85.

This file implements the hardware-dependent parts of the SBLP protocol. It is specific to the Atmel ATTiny85 microcontroller.

Responsibilities of this layer:

  • Interfacing to the bus via a MAX485-style transceiver
  • Byte-level framing with start & stop bits (as per Atmel application note AVR307)
  • Synchronising to the bus on start-up
  • Escaping the synchronisation and escape bytes
